# 🎓 Student Management System

## 📌 Description

This is a Java-based Student Management System developed using NetBeans IDE.
It helps manage student records like name, marks, and grades.

## 🚀 Features

* Add student
* View student
* Update student
* Delete student

## 🛠️ Technologies

* Java
* NetBeans

## ▶️ How to Run

1. Open NetBeans
2. Open project
3. Click Run
